# Pass 1 — Repository Map

Top-level directory classification for `sovereign-music-console` (Sandbox Music). Classifications reflect **2026-07-21** inventory only.

| Directory | Classification | Rationale |
|-----------|----------------|-----------|
| `src/` | **Stable** | Primary React/TypeScript client (~400+ modules). Active development; `sandboxLayer3.tsx` remains central orchestrator per `CODEBASE_HEALTH.md`. |
| `tier34-server/` | **Stable** | Production Sandbox Server (port 3001): locker blobs, search proxy, podcast/DLNA/subsonic routes, acquire workers. Shipped via Docker and desktop anchor mode. |
| `server.ts` | **Stable** | Root UI server (port 3002): Vite dev, catalog proxy, Gemini playlist curation. Built to `dist/server.cjs`. |
| `android/` | **Stable** | Capacitor Android shell: ExoPlayer, Cast, yt-dlp-android natives, custom plugins. Version `0.53.0` / `versionCode 53`. |
| `src-tauri/` | **Stable** | Tauri 2 desktop shell with native audio (symphonia/cpal/wasapi). Release bundles under `target/release/bundle/`. |
| `public/` | **Stable** | Static web assets, PWA icons, community addon manifests (`addons/record-player/`). |
| `scripts/` | **Stable** | Build, F-Droid, Android E2E, locale generation, tier34 build helpers. Referenced from `package.json` scripts. |
| `docs/` | **Stable** | Canonical project documentation (28 topic guides + audit artifacts). |
| `metadata/` | **Stable** | F-Droid submission template (`metadata/fdroid/`). |
| `fastlane/` | **Stable** | Play Store listing text (`metadata/android/en-US/`). Small, maintained adjunct. |
| `docker/` | **Stable** | Sidecar definitions (`demucs-api.py`, referenced by compose). |
| `config/` | **Stable** | Deployment adjunct (`slskd.docker.yml` for Soulseek profile). |
| `assets/` | **Stable** | Branding (`icon.svg`) and `.aistudio` asset stub. |
| `.github/` | **Stable** | CI (`ci.yml`), release (`release.yml`), nightly/phone E2E workflows. |
| `overlay/` | **Evolving** | Self-hosted remote access stack (Headscale, Tailscale, Caddy HTTP/3). Documented in `docs/overlay-network.md` and `docker-compose.overlay.yml`; newer than core app. |
| `src/play/` | **Evolving** | Extracted playback orchestration (queue advance, locker gate, Exo sync). Modular split from `sandboxLayer3.tsx`; still integrating. |
| `src/library/` | **Evolving** | Remote library server browse API (3 modules). Smaller surface; depends on tier34 library routes. |
| `src/tier34/` | **Evolving** | Tier34 HTTP/WebSocket client (`client.ts`, `peerSync.ts`, `connectProtocol.ts`). Growing consumer base across locker, cast, scrobble. |
| `src/addons/` | **Evolving** | User manifest + experimental builtin addon search. Builtin pack gated behind experimental toggle per `ADDONS.md`. |
| `ADDONS.md` (root file) | **Evolving** | Addon manifest workflow; experimental integrations actively documented. |
| `LOCKER_SYNC.md` (root file) | **Evolving** | Cross-device locker sync phased design; Phase 1–3 partially implemented. |
| `proof-screenshots/` | **Experimental** | Screenshots and UI dumps from manual playback QA on physical devices. |
| `proof-oneplus-46349770/` | **Experimental** | Device-specific logcat, player proofs, and UI XML from OnePlus 12 (`46349770`). |
| `_apk_check/` | **Experimental** | APK unzip/analysis scratch (DEX, resources, Python inspect scripts). |
| `dist/` | **Dead** (generated) | Vite client build + `server.cjs`. Regenerated by `npm run build`. |
| `node_modules/` | **Dead** (generated) | npm install output. Not source. |
| `android/app/build/` | **Dead** (generated) | Gradle build intermediates and APK outputs. |
| `android/build/` | **Dead** (generated) | Gradle project build cache. |
| `src-tauri/target/` | **Dead** (generated) | Rust compile artifacts and desktop bundles (source of releases but regenerated). |
| `.idea/` | **Dead** (local) | JetBrains IDE project files; machine-local. |
| `.jdk21/` | **Dead** (local) | Bundled/local JDK 21 for Android builds. |
| `.cursor/` | **Stable** (tooling) | Cursor workspace rules (`rules/no-unsolicited-ui.mdc`). Not app runtime. |
| `STATUS.md` | **Deprecated** | Explicitly deprecated session log; points to `CHANGELOG.md`, `CODEBASE_HEALTH.md`, `docs/CHRONICLE.md`. |
